📌 Dashboard Quick Reference — How to Use This Dashboard
  • WoW thresholds: Changes ≥ ±5% are highlighted green (positive) or red (negative). Smaller changes appear in gray — treat them as essentially flat.
  • Canvas email stats (Sends, Opens, Clicks) are aggregated from step-level data via the Braze canvas/data_summary endpoint. Open rates > 100% can occur when recipients open emails sent in prior periods.
  • Canvas revenue is Braze-attributed and may double-count users enrolled in multiple canvases. Use it for directional comparisons only — Salesforce is the source of truth for revenue.
  • Site Traffic data comes from Google Analytics (GA4) Landing Page report exported weekly to Google Sheets. Session counts may differ slightly from Braze session data due to attribution methodology.
  • Refresh cadence: This dashboard is updated weekly, typically on Mondays, covering the prior Sunday–Saturday period. Canvas entries reflect a 6-day window due to Braze API lag.
